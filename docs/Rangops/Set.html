    module Rangops::Set

    Module defining basic set operations that can be performed on ranges.

    • union

    • intersection

    • relative complement

    • symmetric difference

    along with some convenient aliases and predicates. Loosely follows conventions of Set module from standard library.

    Operations involving 2 ranges require them to overlap to produce result. If the result of operation cannot be expressed as single range, an array of ranges is returned.

def complement(other)
  Rangops::Set.complement(self, other)
end

    Relative complement of 2 ranges. Returns a range covering elements from other that are not covered by self. Returns nil if ranges don’t overlap.

    (1..10).complement(5..15)
=> 10..15

def difference(other)
  Rangops::Set.difference(self, other)
end

    Symmetric difference of 2 ranges. Returns ranges covering elements of both operands, excluding elements common for both of them. Returns nil if ranges don’t overlap.

    (1..10).difference(5..15)
=> [1..5, 10..15]

(11..19) - (15..28)
=> [11..15, 19..28]
